# DDG Captcha Solver (YOLO11)

API-сервис для автоматического распознавания текстовых капч DDoS-Guard на базе YOLO

## Особенности
- Модель обучена на датасете из 533 изображений
- Точность распознавания символов: ~95%
- Автоматический сбор сложных образцов (с низкой уверенностью) для дальнейшего дообучения (вручную потом чтобы обрисовывать)
- Flask-сервер для обработки запросов в реальном времени

## Инструкция
1. Установить зависимости: `pip install ultralytics flask opencv-python torch`
2. Запустить: `python main.py`

## API (POST)
**URL:** `http://0.0.0.0:5000/predict`

**JSON:**
{
  "image": "base64_string"
}

## Переменные
- YOLO_CONF (0.2) — порог детекции
- LOW_CONF_THRESHOLD (0.75) — порог для дообучения
- SAVE_NORMAL_CAPTCHAS (0/1) — сохранение обычных капч
