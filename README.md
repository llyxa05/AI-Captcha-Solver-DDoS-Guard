# DDG Captcha Solver (YOLO11)

API-сервис для автоматического распознавания текстовых капч DDoS-Guard на базе YOLO

## Особенности
- Модель обучена на датасете из 250-500 изображений (не помню точно)
- Точность распознавания символов: ~95%
- Автоматический сбор сложных образцов (с низкой уверенностью) для дальнейшего дообучения (вручную потом чтобы обрисовывать)
- Flask-сервер для обработки запросов в реальном времени

## Требования
- Python 3.x
- Библиотеки: `ultralytics`, `flask`, `opencv-python`, `torch`

## Использование
Запуск сервера:
`python main.py`

Пример запроса:
URL: `http://0.0.0.0:5000/predict`
Метод: POST
Тело (JSON): {"image": "base64_string_here"}

## Переменные окружения
- YOLO_CONF: Порог уверенности детекции (0.2).
- LOW_CONF_THRESHOLD: Порог для сохранения в папку дообучения (0.75).
- SAVE_NORMAL_CAPTCHAS: Сохранение обычных капч (0/1).
